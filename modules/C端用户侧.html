问诊/用药
  • 核心入口：京东App主搜 / 京东健康小程序 / 京东主站健康频道
  • 用户下单 → 等待接单 → 服务完成 → 报告查看 → 问诊/用药
  • 检测业务：两段履约（护士采样 → 实验室检测 → 报告产出）
  • 护理业务：一段履约（护士上门直接完成服务）
二、检测 vs 护理 体验差异
  • 检测业务用户：看到报告页 + 大为AI解读 + 问诊/用药入口，有等待报告环节
  • 护理业务用户：看到服务完成回执，无报告产出环节，服务结束即体验完成
  • 核心差异：检测业务有实验室环节，报告是独立价值交付物
三、六型用户画像
A型 · 急症焦虑型
时效>服务>价格
→ 专送模式（专送不插顺路单，可加价）
B型 · 刚需羊毛型
价格>服务>时效
→ 预约单优先，允许降时效
C型 · 高端体验型
服务>时效>价格
→ VIP服务，服务分优先
D型 · 子女关爱型
服务>价格>时效
→ 子女下单，父母被服务
E型 · 银发经济型
高龄/基础病，特殊关注
→ 服务转介，关注安全性
F型 · 专业复诊型
有明确检测目的
→ 报告解读+用药转化
四、专项课题关联
  • 诊药专项：医·检·诊·药闭环，大为AI承接报告解读，问诊/用药入口切换规则
  • 银发经济：E型用户(高龄/基础病)特殊关注，服务转介，安全性优先
